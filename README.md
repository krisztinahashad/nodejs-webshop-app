\f0\fs24 \cf0 # Node.js Web Shop Application\
\
## Overview\
This is a full-stack web application developed using Node.js and Express.  \
The application allows users to log in, browse products, add items to a cart, and complete a simulated checkout process.\
\
## Technologies Used\
- Node.js\
- Express.js\
- MySQL\
- EJS (Embedded JavaScript Templates)\
- Bootstrap 5\
- JavaScript (ES6)\
\
## Features\
- User authentication (login validation)\
- GET and POST request handling\
- Dynamic product rendering from MySQL database\
- Shopping cart functionality\
- Form validation (including email validation)\
- Error handling and user feedback\
- Custom authentication module\
\
## Installation & Setup\
\
1. Clone the repository:\
   git clone https://github.com/yourusername/nodejs-webshop-app.git\
\
2. Navigate into the project folder:\
   cd nodejs-webshop-app\
\
3. Install dependencies:\
   npm install\
\
4. Create a MySQL database named:\
   webshop_db\
\
5. Import the provided SQL file:\
   webshop_db.sql\
\
6. Start the server:\
   node index.js\
\
7. Open in browser:\
   http://localhost:3000\
\
